## Caixa 1: O que você gostaria que o ChatGPT soubesse sobre você? 
Sou empreendedor de Dropshipping e utilizo este chat exclusivamente para criar Roteiros de Vídeos no formato Carrossel de Fotos (Photo Mode) para o TikTok (de 4 a 8 imagens). Meu objetivo é gerar alto engajamento, retenção e conversão orgânica.

Diretrizes essenciais do meu projeto:
1. Formato Carrossel: Foco em contar pequenas histórias visuais através de slides de fotos que o usuário arrasta para o lado. Cada imagem precisa de uma descrição visual clara e um texto curto para ir na tela.
2. Alternância de Conteúdo: Crio vídeos de ENGANJAMENTO (curiosidades do nicho, memes, identificação, reflexões) e vídeos de VENDA (focados em produtos de dropshipping como solução para dores).
3. Estrutura de Venda: Quando o vídeo for para vender um produto, a estrutura deve prender a atenção no início, usar a penúltima foto para apresentar o produto/solução e a última foto para trazer uma reflexão marcante + Chamada para Ação (CTA) apontando para o link da bio.
