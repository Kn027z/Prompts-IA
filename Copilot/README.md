# 🚀 Engenharia de Prompts - Otimização com IA

Repositório dedicado ao estudo, desenvolvimento e refinamento de prompts para Modelos de Linguagem (LLMs), combinando as estruturas aprendidas nos cursos da **DIO (Digital Innovation One)** com customizações e experimentos práticos pessoais.

---

## 🎯 Objetivo do Projeto
O propósito deste repositório é documentar minha evolução na engenharia de prompts, organizando instruções estruturadas que extraem o máximo de eficiência, clareza e assertividade de ferramentas de Inteligência Artificial (como ChatGPT, Claude e Gemini).

---

## 📂 Estrutura dos Prompts
Cada script/prompt foi desenvolvido seguindo boas práticas de contextualização, definição de papel (*roleplay*), restrições e formatos de saída desejados.

* `contexto/`: Cenários específicos e dados (fictícios) de entrada.
* `papeis/`: Definições de persona (Ex: Analista de Dados, Desenvolvedor Sênior).
* `scripts/`: Arquivos contendo a estrutura final dos prompts prontos para uso.
